## 二、作品摘要（200–400 字）

本作品面向高校辅导员在学生管理与风险研判场景中的“发现—分析—处置—闭环”流程，构建一套集学生画像、预警评分、处置登记、热点洞察与内容策划生成于一体的数智化工作台。系统将学生多源数据分为数值特征与文本内容两条流水线：数值侧采用 KMeans 聚类生成可解释群体标签；文本侧采用 BERTopic 主题建模生成文本主题画像与关键词。基于画像结果，系统计算 0–100 的预警分，并在工作台中提供高风险事件聚合、趋势观察与个体处置登记。为支撑宣传教育与班会组织，系统集成热点榜单（抖音/B站）并提供长文与短视频草稿的流式生成能力，生成过程支持断点续传与历史留痕，且对外部模型调用进行隐私脱敏（不传学号/姓名等直接标识符）。作品可在本地环境快速部署，具备良好的可扩展性与工程化落地基础。

## 三、背景与意义

- **问题背景**：辅导员在日常工作中需要快速识别风险、理解群体结构、生成可用话术/材料并记录处置过程；传统做法依赖人工经验、信息分散、可追溯性弱。
- **作品意义**：通过画像聚类 + 预警评分 + 处置闭环 + 内容生成，提升研判效率与处置规范化；通过脱敏与日志记录增强合规与可控性。

## 四、系统总体设计

### 4.1 总体架构

- **前端工作台**：Vue 3 + TypeScript + Vite  
  - 总览、群体画像、个体画像、内容策划与生成
- **后端服务**：Flask + SQLAlchemy  
  - 数据查询、画像聚类结果读取、热点服务、SSE 流式接口、处置登记接口
- **数据库**：PostgreSQL  
  - 学生信息、数值/文本数据、画像 profile、热点表、内容草稿与生成任务表

### 4.2 核心模块

- **画像与聚类模块**
  - 数值聚类：KMeans（数值特征向量）
  - 文本主题：BERTopic（主题 id + 关键词）
- **预警分模块**
  - 解释性加权：学业/行为/文本/趋势 → 0–100 分
- **处置闭环模块**
  - 个体画像页可登记处置状态与记录（处理中/已闭环/已忽略）
- **热点与内容生成模块**
  - 热点：抖音/B站榜单分页展示，支持跳转原链接
  - 内容：长文/短视频草稿流式生成（SSE），断点续传，历史可追溯

## 五、关键技术与创新点

### 5.1 关键技术

- **多模态画像思路的工程落地**：将学生数据拆分为数值与文本两条管线，分别建模并在同一工作台融合呈现。
- **可解释聚类标签**：聚类结果保存为可读标签（数值簇标签、文本主题标签），便于辅导员理解与使用。
- **SSE 流式输出与断点续传**：生成任务在后台持续运行，前端可断线重连并从偏移量续传。
- **隐私脱敏**：外部模型调用上下文中不包含学号/姓名等直接标识符。

### 5.2 创新点（可按实际调整）

- **“研判—处置—内容”一体化工作流**：不仅展示风险，还提供处置登记与教育内容生成入口，形成闭环。
- **生成过程“留痕可追溯”**：每次点击生成都会创建历史记录，生成中也可从历史返回继续查看。

## 七、运行环境与部署方式

### 7.1 环境要求

- **操作系统**：Windows 10/11（开发验证）；Linux/macOS 亦可
- **Python**：建议使用 conda 环境（见 `backend/environment.yaml`）
- **Node.js**：建议 18+（按本机环境）
- **数据库**：PostgreSQL 12+（建议）

### 7.2 后端启动

进入 `backend/`：

```bash
conda env create -f environment.yaml
conda activate ideological_profiling
setx DATABASE_URL "postgresql+psycopg2://<user>:<password>@127.0.0.1:5432/ideological_profiling_sys"
python -m scripts.init_db
python -m scripts.generate_mock_data
python -m api_server.app
```

### 7.3 前端启动

进入 `frontend/`：

```bash
npm install
npm run dev
```

浏览器访问：`http://127.0.0.1:5173`

## 八、使用说明（评审快速体验）

### 8.1 登录

- 账号/密码：本项目为演示态登录（前端模拟），输入任意非空即可
- 角色：辅导员 / 管理员（演示）

1. **工作台总览**：查看 KPI、群体分布雷达、趋势、高风险预警事件、热点榜单

3. **学生个体画像**：查看预警分与处置状态，填写处置记录并保存
4. **内容策划与生成**：生成长文/短视频草稿，观察流式输出与历史草稿
